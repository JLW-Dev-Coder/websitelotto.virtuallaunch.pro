# canonical-index.html Standard landing page structure for VLP ecosystem sites. ## Required sections 1. Hero — headline, subheadline, primary CTA 2. Value proposition — 3-4 benefit cards 3. How it works — 3 steps 4. Social proof — testimonials or stats 5. Pricing preview — tier summary with link to /pricing 6. FAQ — 4-6 common questions 7. Final CTA — repeat primary call to action 8. Footer — links, legal, contact ## Rules - No external CSS frameworks (use repo's CSS Modules or globals) - Mobile responsive (breakpoints per STYLE.md) - All CTAs link to real routes - No placeholder text — every word is final copy or marked - Brand color from STYLE.md - Jamie L Williams as contact where applicable
